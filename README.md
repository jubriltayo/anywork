# AnyWork

## Introduction
This is a web app where professionals can register andoffer their services to clients.

[anywork](anywork.onrender.com)

## Core Technology
### Installation
It was built using:
* #### django - backend
        pip install django
* #### bootstrap - frontend
        pip install django-bootstrap-v5
* #### postgreSQL - database

### Usage
The project was carried out in a virtual environment

        virtualenv venv
